## 📁 Command Category: Patreon

### 💾 `/patreon`

**Description:** 🔞 Fetches latest posts from a creator on Kemono (NSFW Only).

### 💻 Usage

`/patreon <service> <creator_id>`

### ⚙️ Options

- **`service*`**
  - **Description:** The platform service
  - **Type:** Text
  - **Choices:** `Patreon` (`patreon`), `Fanbox` (`fanbox`), `Gumroad` (`gumroad`), `Boosty` (`boosty`), `Discord` (`discord`)
- **`creator_id*`**
  - **Description:** The unique Creator ID
  - **Type:** Text
