## 📁 Command Category: Globalvoice

### 💾 `/globalvoice`

**Description:** 🔊 Manage global voice settings for this server

### 💻 Usage

`/globalvoice connect <room>`

### 🔧 Subcommands

**`/globalvoice connect <room>`**
> Connect to a global voice room or create a room

**Options for this subcommand:**
- **`room*`**
  - **Description:** Room ID
  - **Type:** Text
