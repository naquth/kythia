## 📁 Command Category: Tempvoice

### 💾 `/tempvoice`

**Description:** 🎧 Manage and customize the Kythia TempVoice system

### 💻 Usage

`/tempvoice remove`
`/tempvoice repair`
`/tempvoice setup [trigger_channel] [category] [control_panel]`

### 🔧 Subcommands

**`/tempvoice remove`**
> Disable the tempvoice system and remove the panel.


**`/tempvoice repair`**
> Repair TempVoice configuration.


**`/tempvoice setup [<trigger_channel>] [<category>] [<control_panel>]`**
> Set up "Join to Create" and send the static control panel.

**Options for this subcommand:**
- **`trigger_channel`**
  - **Description:** Voice channel. (If empty, one will be created automatically)
  - **Type:** Channel
- **`category`**
  - **Description:** Category. (If empty, one will be created automatically)
  - **Type:** Channel
- **`control_panel`**
  - **Description:** Text channel. (If empty, one will be created automatically)
  - **Type:** Channel
